Import Members From Excel - User Guide

This feature has been added to make the process of group member registration faster!

We shall use the MS Excel to record the details of the members

Step 1

  • Open a new Microsoft Excel document

Step 2

  • Save and name the file as "nupola"
  • The first column labelled as "A", should contain the name of the recipients.
  • The second column labelled as "B", should contain the phone numbers of the recipients.
Copy and upload this file to the internal storage of your phone
